Firebase Logic Flow

1. Modul Inventory

Developer input data produk baru di stock/index.html -> Hantar ke koleksi inventory di Firestore.

2. Modul POS

POS tarik data dari koleksi inventory -> Transaksi berlaku -> Simpan rekod ke koleksi sales -> Tolak kuantiti dalam inventory.

3. Dashboard Sync

index.html (Main) menggunakan onSnapshot untuk dengar perubahan real-time dalam koleksi sales & inventory.
